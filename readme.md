* provider 发送消息
* consumer 接受消息

参考[https://spring.io/guides/gs/messaging-rabbitmq/](https://spring.io/guides/gs/messaging-rabbitmq/)

## 安装服务

```
brew innstall rabbitmq

rabbitmq-server
```

## 运行代码

```gradle
./gradlew :gradle-consumer:bootRun 
./gradlew :gradle-provider:bootRun 
 
```
